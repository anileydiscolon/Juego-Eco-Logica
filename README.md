## 🎮 Resumen del Software
Este es un recurso educativo interactivo desarrollado en **HTML5, CSS y JavaScript**. Utiliza la librería de visión artificial **MediaPipe** Para permitir que los usuarios clasifiquen desechos virtuales mediante el movimiento de sus manos frente a una cámara web
## Reglas del juego
Agarras los desechos colocando la mano tipo pinza frente a la camara. Cada vez que aciertan ganan 10 puntos, al equivocarse pierden 5 puntos y si el desecho cae sin ser clasificado les resta 2 puntos.

## 🛠️ Detalles Técnicos
*   **Lenguaje:** JavaScript nativo.
*   **IA:** Modelo de detección de puntos de referencia de la mano (Hand Landmarks).
*   **Calibración:** Sistema de bloqueo de ID para asegurar que el código responda a un solo usuario a la vez, evitando interferencias.
